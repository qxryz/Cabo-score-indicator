# Cabo 计分器

一个用于桌游 Cabo 记分的轻量项目，提供两个可直接使用的版本：

- 桌面版：基于 Python `tkinter` 的本地 GUI 程序
- 网页版：单文件 `index.html`，适合手机或浏览器直接打开使用

仓库里包含了 PyInstaller 打包配置，方便按需自行生成 Windows 可执行文件。

## 项目内容

- `cabo_score.py`：桌面版计分器主程序
- `index.html`：网页版计分器
- `CaboScore.spec`：PyInstaller 打包配置

## 功能说明

- 支持 2 到 4 名玩家
- 自定义玩家名称
- 按轮录入每位玩家的卡牌总和
- 支持 `呼唤 CABO` 结算
- 支持 `神锋特工队 (12,12,13,13)` 特殊结算
- 实时显示当前累计分数
- 达到 100 分时触发一次 `100 -> 50` 的回落规则
- 有玩家超过 100 分后自动结算胜者

## 计分规则

当前项目实现的规则如下：

1. 普通回合：玩家本轮增加自己的卡牌总和。
2. 呼唤 CABO：
   - 若该玩家本轮总和为全场最低，则本轮加 `0` 分。
   - 否则加 `本轮总和 + 10` 分。
3. 神锋特工队：
   - 自己本轮加 `0` 分。
   - 其他所有玩家各加 `50` 分。
4. 当玩家总分首次达到 `100` 分时，会回落到 `50` 分。
5. 当任意玩家总分超过 `100` 分时，游戏结束，未爆分玩家中总分最低者获胜。

## 使用方式

### 方式一：直接运行桌面版

确保本机已安装 Python 3，然后执行：

```bash
python cabo_score.py
```

程序会弹出桌面窗口，选择玩家人数并填写玩家名称后即可开始记分。

### 方式二：直接打开网页版

直接用浏览器打开 `index.html` 即可。

适合：

- 手机浏览器临时记分
- 局域网或文件共享场景
- 不想安装 Python 的使用方式

## 打包桌面版

如果需要重新生成可执行文件，先安装 PyInstaller：

```bash
pip install pyinstaller
```

然后执行：

```bash
pyinstaller CaboScore.spec
```

打包完成后，输出文件默认位于 `dist/` 目录。

## 适用场景

- 线下桌游 Cabo 计分
- 代替纸笔快速记分
- 在桌面端或手机端快速查看领先者与最终胜者
